Desktop Grid

Guideline for how to use layout grids for different devices

Standart Grid

  • Min. Screen Width1440 px
  • Grid Width1224px
  • Gutter Width24 px
  • Column Width80 px
  • Number of Columns12

Responsive Grid

When we need to add sidebar navigation. We should move our workspace and try to not change the workspace size.

When we add side navigation the grid-wide have a responsive size. The left sidebar move the grid to right or left from the center of the screen

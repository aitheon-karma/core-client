Links

Links in UI is one of the most important elements. And we need to follow the rules for link usage

Links usage

Follow 8px margin between text and link if they’re placed on the one line

If not enough place for the link we can transfer it on the next line without additional margins

When link on hover - we need to underline them

We can use links in context. For this usage, we do not need to follow margins in 8px between text and link

Do not transfer link on another line like on the example

Do not divide the link into two separate items.

Links usage

Use accent color for the active links

Follow margins 24 px between links.

Do not accent all links at one time.

Do not underline active links like a tab. It’s not a tabs.

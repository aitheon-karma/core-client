Selection Controls

Selection controls allow users to complete tasks that involve making choices such as selecting options, or switching settings on or off.

Selection controls are found on screens that ask users to make decisions or declare preferences such as settings or dialogs.

Controls type


Checkboxes

Radio buttons

Switches

Checkboxes

A checkbox is a small interactive box that can be toggled by the user to indicate an affirmative or negative choice.

Or it can be used for multiple choice of options like in the polls

Radio buttons

A radio button is a control element that allows the user to choose only one of the proposed option

Switches

A toggle switch represents a physical switch that allows users to turn options on or off.
